# NodeChat
    A chat application built with express and socket.io

## Technologies
* Express - Middleware
* Moustache - Presentation Tier
* Mocha, expect - Testing
* Travis - CI/CD
* Heroku - Hosting

## Usage
    The app is live [here](https://sai-chat.herokuapp.com)
